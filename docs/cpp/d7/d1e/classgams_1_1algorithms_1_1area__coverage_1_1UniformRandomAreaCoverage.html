void gams::algorithms::area_coverage::UniformRandomAreaCoverage::generate_new_position ( void  )
protectedvirtual

Generate new next position.

Random positions are generated in the bounding rectangle of the region and checked if it is actually in the region.

In theory this could never happen when given any arbitrary region. The average selection time is area(bounding_box) / area(region). In practice, this finds a valid position quickly enough to not affect the system.
