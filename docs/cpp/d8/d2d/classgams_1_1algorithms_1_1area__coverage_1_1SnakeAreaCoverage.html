Compute waypoints.

The endpoints of each parallel lines are found and stored in order.

Assuming the bounding area is convex, only one of these loops will actually do work, the other will fail out the first time through the while loop due to not finding an intercept.

The working loop will find all the snaking waypoints

Check each edge for intercepts Since we are assuming this is a convex polygon, it can have at most two intercepts

if potential edge is not a vertical line, then find intercept if potential edge is vertical, then it cannot intercept
