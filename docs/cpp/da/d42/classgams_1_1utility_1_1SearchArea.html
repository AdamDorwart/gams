gams::utility::Region gams::utility::SearchArea::get_convex_hull ( ) const

Find the convex hull.

Returns
Convex hull of the regions

completely rewritten by James Edmondson on 4/3/2016. old version harder to maintain, more computationally expensive and unlikely to produce convex hulls consistently due to modifications made using arbitrary insertion of [0,0,0] origins with cross products.

The new convex hull uses Andrew's Monotone Chain Algorithm rather than Graham scan and also inlines the cross

memory allocation is expensive, so only do that once when creating the list of all points
